# HTML semântico

```
<header> - Cabeçalho da pagina ou de determinada sessão
<main> - Bloco principal da pagina, onde estará o conteudo principal
<footer> - Rodapé da pagina ou de determinada sessão
<article> - Artigo da pagina,com textos e até cards.
<aside> - Sempre será utilizado quando necessário apresentar informaçes relevantes dos blocos em volta dele
<section> - Sessão ou grupo de informaçes na tela
<nav> - Sessão de navegação, links principais da pagina
<figure> - Envolverá normalmente uma imagem na tela
<time> - Apresenta de forma estruturada data e hora
```
